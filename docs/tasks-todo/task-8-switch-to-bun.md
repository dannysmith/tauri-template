# Switch to Bun

## Overview

Replace npm with Bun as the package manager.

## Key Areas

### Package Management Only

- Use Bun for package management only
- Do NOT use Bun for build processes (keep Vite/Tauri build system)
- Literally just replacing npm with bun for installing/managing packages

### Migration

- Replace package-lock.json with bun.lockb
- Update any scripts referencing npm
- Update documentation
